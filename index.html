1.¿Qué es un Dato?

Un dato es una representación simbólica (numérica, alfabética, alfanumérica, etc.) de un atributo o una característica de una entidad, objeto o fenómeno.

2.¿Qué es información?

La información es el resultado del procesamiento, organización y análisis de datos, que proporciona significado y contexto para tomar decisiones o realizar acciones.

3.¿Diferencia entre dato e información?

La diferencia principal radica en que los datos son la materia prima que se recopila, mientras que la información es el resultado del procesamiento de esos datos, que los convierte en algo significativo y útil.

4.¿Tipos de datos?

Los tipos de datos pueden variar dependiendo del contexto, pero algunos tipos comunes incluyen datos numéricos, datos alfanuméricos, datos de fecha y hora, datos booleanos, entre otros.

5.Qué es un modelado de Datos?

El modelado de datos es el proceso de definir y organizar la estructura de una base de datos de manera que refleje con precisión las características del negocio o del sistema que se está diseñando.

6.¿Tipos de bases de Datos, según:

Según la variabilidad, las bases de datos pueden ser estáticas o dinámicas. Según el contenido, pueden ser bases de datos operativas, analíticas o mixtas. Según el modelo de base de datos, pueden ser relacionales, NoSQL, jerárquicas, de red, entre otros.

7.¿Qué es el modelo relacional?

El modelo relacional es un modelo de datos que organiza los datos en tablas relacionadas entre sí mediante claves primarias y claves foráneas.

8.¿Describa los elementos del Diagrama entidad Relacional?

Los elementos del Diagrama Entidad-Relación (DER) incluyen entidades, atributos y relaciones. Las entidades se representan mediante rectángulos, los atributos se muestran dentro de las entidades y las relaciones se representan mediante líneas que conectan las entidades.

9.¿Qué es una entidad?

Una entidad es un objeto o concepto del mundo real que puede ser identificado y del cual se puede recopilar información.

10.¿Qué es un atributo y sus tipos?

Un atributo es una propiedad o característica de una entidad que ayuda a describirla. Los tipos de atributos incluyen atributos simples, compuestos, monovalentes y multivaluados.

11.¿Qué es una relación?

Una relación es una asociación o conexión entre dos o más entidades en una base de datos.

12.¿Qué son las llaves primarias, foránea y sus características?

Las llaves primarias son atributos que identifican de manera única a cada registro en una tabla. Las llaves foráneas son atributos que establecen una relación entre dos tablas, utilizando la llave primaria de una tabla como referencia en otra. Las características de las llaves incluyen unicidad, no nulidad y consistencia.

13.¿Qué son las reglas del Negocio?

Las reglas del negocio son restricciones o directrices que determinan cómo deben manipularse los datos en una base de datos para garantizar la integridad y consistencia de la información.

14. ¿Qué es la cardinalidad o mapeo?

La cardinalidad o mapeo en el contexto de las bases de datos describe la relación entre las instancias de una entidad y las instancias de otra entidad en una relación. Puede ser uno a uno, uno a muchos o muchos a muchos.
